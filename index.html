Chakra UI Theme Editor

Customize your Chakra UI theme with this interactive editor. Modify colors, typography, spacing, and more, then export your theme file.

There are great starter color palettes for inspiration. We have created reusable UI components in your colors like cards and tables that you can use in your site as-is.

Theme Colors

Create and manage color palettes for your theme. Add colors manually or extract them from images.

No Google Fonts API key found in environment variables. Please add a GOOGLE_FONTS_API_KEY entry to your .env file.

Quick setup instructions:

1. Create a .env file in the project root

2. Add: GOOGLE_FONTS_API_KEY=your_api_key_here

3. Restart the development server
